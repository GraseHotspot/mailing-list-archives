ok my setup is very simple:

- Nanostation M5 in wifi AP mode (hidden ssid), connected by cable to the 
grase box switch. The M5 has networking is bridge with STATIC IP on 
10.1.0.2 (gw is the grase box is 10.1.0.1)
-Nanostation M5 in station mode, networking is bridge with STATIC IP 
10.1.0.3 and gw 10.1.0.1

if I connect manually (i.e. connecting the first antenna to my laptop with 
fixed ip 10.1.0.33) I can ping both antennas and also ALL the devices 
connected to the M5 station (remote antenna)

if I connect my laptop to the second nanostation (by cable) I can ping the 
first antenna (so the link works, and my air quality index is 99%, 2ms 
latency ping)

but if I try to ping 10.1.0.1 (grase box) I cannot even ping!!!
:-(
